# Florida School-Based SLP Salary Comparison

**Live site:** [FloridaSchoolSLPSalaries.com](https://floridaSchoolSLPSalaries.com)

A salary transparency tool for district-employed speech-language pathologists across all 67 Florida school districts — built by SLPs, for SLPs.

---

## What This Tool Does

- 💰 **Salary Calculator** — Look up Year 1 starting salary by district, degree level, and CCC status
- ⚖️ **Compare Districts** — Side-by-side salary breakdown for any two districts
- 📊 **All Districts** — Sortable, searchable table of all 67 Florida districts
- 🔍 **District Profile** — Full salary breakdown, contract details, and cost of living data
- 📬 **Submit a Correction** — Community-sourced data corrections reviewed before publishing

---

## Important Notes

> ⚠️ **District-hired SLPs only.** This tool covers SLPs employed directly by school districts. It does NOT include contract or agency-based positions.

Data is sourced from publicly available salary schedules, verified where possible. Confidence levels are shown for each district (✅ Verified / ⚠️ Estimated).

---

## How It Works

- The frontend (`index.html`) is a single-file static site hosted on GitHub Pages
- Data is stored in a connected Google Sheet and served via Google Apps Script
- Corrections submitted through the site are reviewed by the maintainers before any changes are made

---

## Data Coverage

- **School year:** 2025–2026
- **Districts:** All 67 Florida school districts
- **Last major update:** March 2026
